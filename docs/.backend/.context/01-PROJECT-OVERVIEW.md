# 01. 프로젝트 개요 — 모니: 모의 머니

> 원본: 기획 문서(노션) 0~3번 섹션을 정리. 와이어프레임/링크 등 시각 자료는 원본 노션 문서를 참고하세요.

## 주제

초보자를 위해 복잡한 금융 지표·뉴스를 AI로 알기 쉽게 제공하고, 개인 맞춤형 포트폴리오 분석과
실시간 주식 거래를 지원하는 **모의 투자 플랫폼**.

## 목표

- 경제 지식이 없어도 이해하기 어려운 거시 지표·기업 이슈를 AI가 요약해 초보자도 쉽게 이해할 수 있도록 제공
- 사용자의 투자 성향·선호 방식을 측정하고, 이에 맞춘 포트폴리오 분석·정보 제공
- 실제 시장가와 연동되는 거래 경험으로, 자산 손실 위험 없는 매수/매도 경험 제공
- 구독 기반 AI 포트폴리오 분석 기능으로 수익화 구조 구현
- 고가용성·동시성 환경에서도 안정적인 주가 정보 제공, 거래 서비스 구현

---

## 서비스 기능 정의

### 1. 유저 서비스
- 회원가입 및 로그인
- 회원 정보 수정
- 개인 주식 성향 측정 및 관심사 설정 기능
- AI 분석 구독(결제)
- 맞춤형 홈 피드

### 2. 포트폴리오
- 포트폴리오 대시보드
  - 포트폴리오 수익률 계산
  - 보유 종목 현황

### 3. 모의 투자 서비스
- **종목 매수**
- **종목 매도**
- 종목 조회
- 구매 종목 보기
- 즐겨찾기 종목 보기
- 주식 모으기(도전)
- 자동 손절/익절 기능(도전)
- 사용자 수익 랭킹(도전)

### 4. 투자 종목 조회 서비스
- 종목 검색 서비스
- 종목 기본 정보 상세 조회 서비스
- **실시간 주가, 거래 데이터 조회**
- **실시간 투자 인기 트렌드, 종목 제공 기능**

### 5. AI 서비스
- **(실시간 주식 데이터 기반 RAG) AI 기업 이슈 분석 및 뉴스 요약**
- 주가 상승/하락 이유
- AI 포트폴리오 분석

### 6. 알림 서비스
- 사용자 맞춤 알림 서비스 (예: 미장 개장 10분 전)

---

## 기능 우선순위

### P0 (필수, 최우선 구현)
1. **유저 서비스**: 회원가입 및 로그인
2. **포트폴리오**: 포트폴리오 수익률 계산, 보유 종목 현황
3. **모의 투자 서비스**: 종목 매수, 종목 매도, 구매 종목 보기
4. **투자 종목 조회 서비스**: 종목 검색, 종목 기본 정보 상세 조회, 실시간 주가/거래 데이터 조회
5. **AI 서비스**: (실시간 주식 데이터 기반 RAG) AI 기업 이슈 분석 및 뉴스 요약

### P1
1. **유저 서비스**: 회원 정보 수정, 개인 주식 성향 측정 및 관심사 설정, 맞춤형 홈 피드
2. **모의 투자 서비스**: 즐겨찾기 종목 보기
3. **투자 종목 조회 서비스**: 실시간 투자 인기 트렌드/종목 제공 기능
4. **알림 서비스**: 사용자 맞춤 알림 서비스 (예: 미장 개장 10분 전)

### P2
1. **유저 서비스**: AI 분석 구독
2. **모의 투자 서비스**: 주식 모으기(도전), 자동 손절/익절(도전), 사용자 수익 랭킹(도전)
3. **AI 서비스**: 주가 상승/하락 이유, AI 포트폴리오 분석

> Claude에게 새로운 기능 구현을 요청할 때는 위 우선순위(P0 → P1 → P2) 순서를 기본 가이드로
> 삼되, 사용자가 명시적으로 다른 항목을 지정하면 그것을 우선합니다.
